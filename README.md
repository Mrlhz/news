## 新闻项目

## 在虚拟环境中使用pip命令生成依赖包文件：
```
(venv) $ pip freeze > requirements.txt
```
## 当需要创建这个虚拟环境的完全副本，可以创建一个新的虚拟环境，并在其上运行以下命令：
```
(venv) $ pip install -r requirements.txt
# 豆瓣源加速
pip install -r requirements.txt -i https://pypi.doubanio.com/simple/
```
